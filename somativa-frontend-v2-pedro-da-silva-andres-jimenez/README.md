# 🏃‍♂️ FitTrack

Este é um projeto desenvolvido para a avaliação de Front-End, focado em criar interatividade para um site relacionado a saúde das pessoas, como registrar quantidade de água e alimentação.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
Foi organizado através de pastas e subpastas:

/
├── index.html
└── src/
    ├── styles/  (Estilos da página)
    ├── scripts/ (Lógica de interação)
    └── images/  (Ativos visuais)

## 🚀 Funcionalidades Implementadas

* **Dia de Jejum (Limpeza com Aviso Visual):** Texto para o atleta esvaziar a lista e voltar no outro dia.
* **Recepção do Atleta (Date e Condicionais):** Texto de recpção para o atleta de acordo com o horário de seu dia.
* **Calculadora de Hidratação (Input com Tratamento):** Cálculo matemático para o atleta medir as calorias de sua refeição. 
* **O Diário de Refeições (Click com Emojis):** Registro de refeições que o atleta fez em seu dia.
* **Banner Motivacional (Eventos de Mouse):** Foco para o atleta.
